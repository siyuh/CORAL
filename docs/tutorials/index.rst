CORAL Tutorial
--------------

End-to-end walkthrough of the CORAL model using mouse thymus data with
CODEX (high-resolution protein) and Visium (low-resolution RNA). Covers data
loading, preprocessing, model creation, training, inference, and evaluation.

Runtime: ~15 minutes on GPU.

Slide Alignment Tutorial
-------------------------

Semi-automated alignment of multi-omics tissue slides using BRISK keypoints
and VGG descriptors. Covers image preprocessing, feature detection, RANSAC
transformation estimation, and coordinate transformation.
